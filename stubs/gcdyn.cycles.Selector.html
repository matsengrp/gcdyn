gcdyn.cycles.Selector

class gcdyn.cycles.Selector[source]

Bases: ABC

A class for GC selectors, which determine the fitness for a list of nucleotide sequences. Selectors return a list of tuples that can be interpreted by a proliferator method

Methods

Assigns the fitness for each sequence, with normalization depending on whether competition is considered.

Parameters
