gcdyn.fitness

Uses phenotype to determine fitness of B cells in the germinal center light zone.

Classes

Fitness

Class to determine fitness from phenotype for a collection of sequences.
class gcdyn.fitness.Fitness(tfh_function, concentration_antigen=1e-10)[source]

Class to determine fitness from phenotype for a collection of sequences.

Parameters

  • tfh_function – method to map from KD to T cell help, taking in a list of \(K_D\) values and producing a list of absolute T cell help quantities

  • concentration_antigen – molar concentration of antigen to determine antigen bound

linear_fitness(kd_values, y_intercept=0, slope=1)[source]

Combines methods to get the antigen bound and T cell help from the KD using a linear model.

Parameters

  • kd_values – list with KD value for each sequence

  • y_intercept – amount of T-cell help with no antigen bound

  • slope – coefficient of relationship between antigen bound and T cell help

Returns

list with non-normalized T cell help value for each sequence

Return type

t_cell_help_values

Combines methods to get the antigen bound and T cell help from the KD using a sigmoidal model.

Parameters

  • kd_values – list with KD value for each sequence

  • maximum_Tfh – maximum possible T cell help generated by model

  • curve_steepness – logistic growth rate of T cell help

  • midpoint_antigen_bound – value of input antigen bound to set as midpoint

Returns

Return type

normalized_fitness_df(seq_list, calculate_KD)[source]

Produces a dataframe including the fitness of a series of sequences given KD values.

Parameters

  • seq_list – list of DNA sequences

  • calculate_KD – method that produces a KD value for each sequence in the list

Returns

DataFrame with columns frac_antigen_bound, KD, t_cell_help, and normalized_t_cell_help with t_cell_help values based on tfh_function set on initialization
