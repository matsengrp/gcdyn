gcdyn.cycles.GC

class gcdyn.cycles.GC(sequence, proliferator, mutator, selector, N0=1, Nmax=None, rng=Generator(PCG64) at 0x7FAA0F84B660)[source]

Bases: object

A class for simulating a germinal center with discrete LZ \(\leftrightarrow\) DZ cycles.

Parameters

  • sequence (str) – root nucleotide sequence

  • proliferator (Callable[[TreeNode, float, Generator], None]) – neutral tree generator for DZ proliferation and mutation. This tree is assumed to run for exactly one unit of simulation time, but this is not enforced (if not, you may generate non-ultrametric trees). If it produces extinct leaves, they should be marked with a node attribute node.terminated = True.

  • mutator (Callable[[str, float, Generator], str]) – mutation generator that takes a starting sequence and an exposure time and returns a mutated sequence

  • selector (Callable[[List[str]], List[Any]]) – takes a list of sequences and returns their fitness parameters

  • N0 (int) – initial naive abundance

  • Nmax (Optional[int]) – population capacity. If not None and the number of alive cells exceeds Nmax in any cycle, the population will be randomly downsampled to size Nmax.

  • rng (Generator) – random number generator
