gcdyn.cycles.simple_proliferator

gcdyn.cycles.simple_proliferator(treenode, cell_divisions, dist=None, rng=Generator(PCG64) at 0x7F4628EDB3C0)[source]

Recursively populates descendants on a tree node based on the number of integer cell divisions indicated. Branch lengths are set to 1/cell_divisions, such that the distance from the root to all leaves will be 1.

Parameters

  • treenode (TreeNode) – root node to populate from

  • cell_divisions (int) – number of cell divisions from the cell represented by treenode

  • dist (Optional[float]) – distance from each parent to child node

  • rng (Generator) – random number generator

Return type

None
