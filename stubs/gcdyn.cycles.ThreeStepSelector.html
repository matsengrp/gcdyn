gcdyn.cycles.ThreeStepSelector

class gcdyn.cycles.ThreeStepSelector(igh_frame=1, igk_frame=1, igk_idx=336, naive_sites_path='https://raw.githubusercontent.com/jbloomlab/Ab-CGGnaive_DMS/main/data/CGGnaive_sites.csv', model_path='Linear.model', tdms_phenotypes=['delta_log10_KD', 'delta_expression'], log10_naive_KD=-10.43, concentration_antigen=1e-09, total_t_cell_help=50, max_help=2, antigen_frac_limit=0.2)[source]

Bases: Selector

A Selector that uses DMS data as well as a discrete amount of T cell help that is assigned.

Initializes values for a DMSPhenotype to calculate KD based on sequence from torchDMS model if KDs are not provided, and sets antigen concentration for determining antigen bound.

Parameters

  • igh_frame (int) – frame for translation of Ig heavy chain

  • igk_frame (int) – frame for translation of Ig light chain

  • igk_idx (int) – index of Ig light chain starting position

  • naive_sites_path (str) – path to CSV lookup table for converting from scFv CDS indexed site numbering to heavy/light chain IMGT numbering

  • model_path (str) – path to torchdms model for heavy and light chain

  • tdms_phenotypes (List[str]) – names of phenotype values produced by passed-in torchdms model (delta_log10_KD expected as a phenotype)

  • log10_naive_KD (float) – KD of naive Ig

  • concentration_antigen (float) – molar concentration of antigen to determine antigen bound

  • total_t_cell_help (float) – total units of T cell help to be distributed in germinal center

  • antigen_frac_limit (float) – lower limit (inclusive) for the fraction antigen bound to be returned.
