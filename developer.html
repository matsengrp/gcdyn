Developer tools#

Developer install:

make install

Warning

Pip installation of ETE’s PyQt5 dependency has been found to fail (with an error like this) on ARM Mac. You will need to install PyQt5 (e.g. with Conda), and then try make install-no-pyqt instead of the command above.

Run tests:

make test

Format code:

make format

Lint:

make lint

Build docs locally (you can then see the generated documentation in docs/_build/html/index.html):

make docs

Docs are automatically deployed to github pages via a workflow on push to the main branch.
