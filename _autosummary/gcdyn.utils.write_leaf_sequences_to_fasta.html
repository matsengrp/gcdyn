gcdyn.utils.write_leaf_sequences_to_fasta

gcdyn.utils.write_leaf_sequences_to_fasta(tree, file_path, naive=None)[source]

Write the sequences at the leaves of a tree to a FASTA file, potentially including a naive sequence.

Parameters:

  • tree (TreeNode) – A tree with sequences at the leaves.

  • file_path (str) – The path to the FASTA file to write.

  • naive (Optional[bool]) – Flag to include a naive root sequence in output.
