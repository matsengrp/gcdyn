class gcdyn.cycles.GC(sequence, proliferator, mutator, selector, N0=1, Nmax=None, rng=Generator(PCG64) at 0x7F0BCB7FABA0)[source]

A class for simulating a germinal center with discrete LZ \(\leftrightarrow\) DZ cycles.

Parameters

  • sequence (str) – root nucleotide sequence

  • proliferator (Callable[[TreeNode, float, Generator], None]) – neutral tree generator for DZ proliferation and mutation. This tree is assumed to run for exactly one unit of simulation time, but this is not enforced (if not, you may generate non-ultrametric trees). If it produces extinct leaves, they should be marked with a node attribute node.terminated = True.

  • mutator (Mutator) – mutation generator that takes a starting sequence and an exposure time and returns a mutated sequence

  • selector (Selector) – takes a list of sequences and returns their fitness parameters

  • N0 (int) – initial naive abundance

  • Nmax (Optional[int]) – population capacity. If not None and the number of alive cells exceeds Nmax in any cycle, the population will be randomly downsampled to size Nmax.

  • rng (Generator) – random number generator

gcdyn.cycles.binary_proliferator(treenode, p, *args, rng=Generator(PCG64) at 0x7F0BCB7FAC80)[source]

Binary dark zone step (Galton-Watson). With probability \(1-p\) the input’s terminated attribute is set to True, indicating extinction. With probability \(p\) add two children to the input with terminated attributes set to False, indicating birth.

Parameters

  • treenode (TreeNode) – root node to simulate from

  • p (float) – bifurcation probability \(p \in [0, 1]\)

  • rng (Generator) – random number generator

gcdyn.cycles.cell_div_balanced_proliferator(treenode, cell_divisions, rng=Generator(PCG64) at 0x7F0BCB7FAE40)[source]

Populates descendants on a tree node based on the number of cell divisions indicated, producing the number of children expected by \(2^{cell\_divisions}\), where cell_divisions is not necessarily an integer value. A full binary tree is generated, then leaf nodes are removed at random from the set of alternating leaf nodes until there are the expected number of children.

Parameters

  • treenode (TreeNode) – root node to populate from

  • cell_divisions (float) – number of cell divisions from the cell represented by treenode

  • kd – calculated KD value for sequence

  • fitness – calculated fitness for sequence

  • rng (Generator) – random number generator

Return type

None
