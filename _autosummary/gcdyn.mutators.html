gcdyn.mutators#

Mutation effects generators ^^^^^^^^^^^^^^^^^^^^^^^^^^^

Abstract base class for defining generic mutation effect generators (i.e. \(\mathcal{p}(x\mid x')\)), with arbitrary ete3.TreeNode attribute dependence. Some concrete child classes are included.

Classes

AttrMutator

Abstract base class for mutators that mutate a specified ete3.TreeNode node attribute.

ContextMutator

Class for hotspot-aware mutation model using mutability substitution probabilities expressed in terms of context.

DiscreteMutator

Mutations on a discrete space with a stochastic matrix.

GaussianMutator

Gaussian mutation effects on a specified attribute.

KdeMutator

Gaussian kernel density estimator (KDE) for mutation effect on a specified attribute.

Mutator

Abstract base class for generating mutation effects given ete3.TreeNode object, which is modified in place.

SequenceMutator

Mutations on a DNA sequence.

SequencePhenotypeMutator

Mutations on a DNA sequence that get translated into a functional phenotype.

UniformMutator

Uniform mutation process.
