gcdyn.mutators.ContextMutator

class gcdyn.mutators.ContextMutator(mutability, substitution, seq_to_contexts)[source]

Bases: SequenceMutator

Class for hotspot-aware mutation model using mutability substitution probabilities expressed in terms of context.

Parameters:

  • mutability (Series) – Mutability values for each local nucleotide context.

  • substitution (DataFrame) – Table of nucleotide substitution bias (columns) for each local nucleotide context (index).

  • seq_to_contexts (Callable[[str], list[str]]) – A function that accepts a sequence and splits it into local contexts.
