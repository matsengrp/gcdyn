gcdyn.poisson.SequenceContextMutationResponse#

class gcdyn.poisson.SequenceContextMutationResponse(mutability, seq_to_contexts, mutation_intensity=1.0, grad=False)[source]#

Bases: SequenceResponse

A Response that accepts a sequence and outputs an aggregate mutation rate.

Importantly, the mutability needs to be in units of mutations per unit time.

Parameters:

  • mutability (Series) – a mapping from local context to mutation rate (mutations per site per unit time)

  • seq_to_contexts (Callable[[str], list[str]]) – a function that accepts a sequence and splits it into local contexts

  • mutation_intensity (float) – a scaling factor for the mutability

  • grad (bool) – See Response.

Λ(node, Δt)#

Evaluate the Poisson intensity measure of the time interval \([t, t+\Delta t)\), defined as.

\[\Lambda(t, t+Δt) = \int_{t}^{t+\Delta t} \lambda(t')dt',\]

for a tree node at time \(t\). This is needed for sampling waiting times and evaluating the log probability density function of waiting times.

Parameters:

  • node (TreeNode) – The node whose state is accessed to evaluate the response function.

  • Δt (float) – Time interval duration (Lebesgue measure).

Return type:

float

Λ_inv(node, τ)#

Evaluate the inverse function wrt \(\Delta t\) of Response.Λ(), \(\Lambda_t^{-1}(\tau)\), such that \(\Lambda_t^{-1}(\Lambda(t, t+\Delta t)) = \Delta t\). This is needed for sampling waiting times. Note that \(\Lambda_t^{-1}\) is well-defined iff \(\lambda(t) > 0 \forall t\).

Parameters:

  • node (TreeNode) – The node whose state is accessed to evaluate the response function.

  • τ (float) – Poisson intensity measure of a time interval.

Return type:

float
